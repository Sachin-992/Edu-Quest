# EDUCORE-OMEGA: Mobile UX Guide

> **Document Version**: 1.0  
> **Last Updated**: January 2026  
> **For**: All Users (Teachers, Parents, Admins)

---

## Mobile-First Design

EDUCORE-OMEGA is designed for the Indian school context where:
- Teachers use smartphones in classrooms
- Parents check grades on phones
- Desktops are primarily for admin work

---

## Mobile vs Desktop Differences

| Feature | Mobile | Desktop |
|---------|--------|---------|
| Navigation | Hamburger menu | Sidebar |
| Tables | Horizontal scroll | Full width |
| Forms | Single column | Multi-column |
| Buttons | Full width | Inline |
| Dashboard cards | Stacked | Grid |

---

## Mobile Navigation

### Opening Menu
1. Tap hamburger icon (☰) in top-left
2. Menu slides in from left
3. Tap any menu item
4. Menu closes automatically

### Closing Menu
1. Tap outside menu area
2. Or tap X button
3. Or swipe left

---

## Role-Specific Mobile Tips

### Teachers on Mobile

**Best For:**
- ✅ Quick attendance marking (tap P/A/L)
- ✅ Viewing class lists
- ✅ Checking schedules

**Challenging On Mobile:**
- ⚠️ Entering multiple marks (use tablet/desktop)
- ⚠️ Uploading large files

**Tips:**
- Turn phone sideways for tables
- Use "Save Draft" frequently
- Pre-fill data on desktop if possible

### Parents on Mobile

**Best For:**
- ✅ Checking attendance
- ✅ Viewing marks
- ✅ Reading notices

**Tips:**
- Bookmark app on home screen
- Swipe to see more columns
- Scroll down for full data

### Admins on Mobile

**Best For:**
- ✅ Viewing dashboards
- ✅ Quick approvals
- ✅ Checking logs

**Challenging On Mobile:**
- ⚠️ Creating users (long forms)
- ⚠️ Bulk operations

**Recommendation:** Use desktop for admin work when possible.

## Table Scrolling

When tables are too wide:

1. Touch the table and swipe left/right
2. Scroll indicator appears at bottom
3. Column headers stay visible

---

## Known Mobile Limitations

| Limitation | Workaround |
|------------|------------|
| Long forms hard to fill | Use desktop |
| PDF download may open in browser | Use built-in Share function |
| Some charts truncated | Turn phone sideways |
| File upload slow on 3G | Use WiFi |

## Offline Behavior

EDUCORE-OMEGA requires internet connection:

| Scenario | Behavior |
|----------|----------|
| No internet | Error message, cannot proceed |
| Slow internet | Loading indicators, may timeout |
| Internet restored | Refresh page to continue |

**Important:** Data is NOT saved locally. Always ensure connection before saving.

---

## Touch Targets

All buttons and links are sized for touch:
- Minimum 44px height
- Adequate spacing between items
- Clear visual feedback on tap
