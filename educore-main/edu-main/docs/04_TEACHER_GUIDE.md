## Getting Started

### Your Login Credentials

- **Username**: Your email address (provided by admin)
- **Password**: Your date of birth in DDMMYYYY format
  - Example: Born March 15, 1985 → Password: `15031985`

### First Login

1. Go to application URL
2. Enter your email
3. Enter your DOB password
4. You will be prompted to change your password
5. Create a new secure password
6. Save it securely

## Daily Workflows

### 1. Mark Attendance

**Step-by-step:**

1. Login to Teacher Portal
2. Click **Attendance** tab
3. Select your class from dropdown
4. Select today's date
5. For each student:
   - Click **Present** (✓)
   - Click **Absent** (✗)
   - Click **Late** (⏰)
6. Click **Save Attendance**
7. **SUCCESS**: See confirmation message

**Important:**
- Attendance is linked to your user ID
- You can only modify today's attendance
- Past attendance requires admin intervention

---

### 2. Enter Marks/Grades

**Step-by-step:**

1. Click **Marks Entry** tab
2. Select class
3. Select subject
4. Select exam type (Unit Test, Mid-term, Final)
5. For each student:
   - Enter marks obtained
   - Grade auto-calculates (A+, A, B+, B, C, F)
6. Click **Save Marks**
7. **SUCCESS**: Marks saved to database

**Grading Scale:**
| Marks | Grade |
|-------|-------|
| 90-100 | A+ |
| 80-89 | A |
| 70-79 | B+ |
| 60-69 | B |
| 50-59 | C |
| Below 50 | F |

---

### 3. Create Assignment

**Step-by-step:**

1. Click **Assignments** tab
2. Click **+ New Assignment**
3. Fill in:
   - Title (required)
   - Description
   - Class (required)
   - Subject
   - Due Date (required)
   - Type (Homework/Project/Test)
4. Click **Create**
5. **SUCCESS**: Assignment visible to students

---

### 4. Upload Study Material

**Step-by-step:**

1. Click **Resources** tab
2. Click **Upload File**
3. Select file from device (PDF, DOC, images)
4. Select target class
5. Add description
6. Click **Upload**
7. **SUCCESS**: File available for students

**File limits:**
- Max size: 10MB per file
- Supported: PDF, DOC, DOCX, PPT, PPTX, JPG, PNG

---

### 5. View Student List

**Step-by-step:**

1. Click **My Classes** tab
2. Select class from dropdown
3. See list of students with:
   - Name
   - Roll number
   - Attendance percentage
   - Average marks

## What You CANNOT Do

| Action | Reason |
|--------|--------|
| Create new students | Admin-only |
| Delete students | Admin-only |
| View other teachers' classes | Access restricted |
| Modify past attendance | Data integrity |
| View fee records | Admin/Parent only |
| Access audit logs | Admin only |

## Mobile Usage

- All teacher features work on mobile
- Best for quick attendance marking
- Larger forms easier on tablet/desktop
- Portrait mode: single column layout
- Landscape mode: table view available

**Tips for mobile:**
- Turn phone sideways for marks entry
- Use quick-tap for attendance (P/A/L buttons)

## Error Messages

| Error | Meaning | Fix |
|-------|---------|-----|
| "Invalid credentials" | Wrong password | Use DOB format DDMMYYYY |
| "No classes assigned" | Admin hasn't assigned classes | Contact admin |
| "Permission denied" | Trying to access restricted area | Stay in your portal |
| "Save failed" | Network issue | Retry, check internet |
